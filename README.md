# 🤖 Telegram-бот приёмной комиссии 2026/2027

Бот для абитуриентов Филиала АГТУ в Ташкентской области Республики Узбекистан.
Работает 24/7, отвечает на вопросы о приёме (сроки, стоимость, направления,
вступительные испытания, документы) с помощью ИИ по базе знаний `knowledge.md`,
имеет меню с кнопками и админ-команды (статистика, рассылка, обновление базы).

Всё необходимое — бесплатно. Платным может быть только ИИ-сервис, но есть бесплатные тарифы (Groq).

---

## Что входит

| Файл | Назначение |
|------|------------|
| `bot.py` | Код бота |
| `knowledge.md` | База знаний — **редактируйте этот файл, чтобы менять ответы** |
| `requirements.txt` | Список библиотек |
| `.env.example` | Шаблон настроек (скопируйте в `.env`) |

---

## Шаг 1. Создать бота в Telegram
1. Откройте **@BotFather** в Telegram → команда `/newbot`.
2. Задайте имя и username бота.
3. Скопируйте полученный **токен** (вида `123456789:AA...`).

## Шаг 2. Узнать свой Telegram ID (для админки)
Напишите боту **@userinfobot** — он пришлёт ваш числовой `id`.

## Шаг 3. (Опционально) Получить бесплатный ИИ-ключ
Для «умных» ответов своими словами:
1. Зайдите на **https://console.groq.com** (бесплатная регистрация).
2. Создайте **API Key**.
3. Вставьте его в `.env` в поле `AI_API_KEY`.

> Без ключа бот тоже работает — отвечает поиском по разделам базы. Но с ИИ ответы заметно лучше.
> Условия бесплатных тарифов у ИИ-сервисов со временем меняются — проверяйте актуальные лимиты.

## Шаг 4. Настроить
1. Скопируйте `.env.example` → `.env`.
2. Заполните `BOT_TOKEN`, `ADMIN_IDS`, при желании `AI_API_KEY`.

## Шаг 5. Запустить локально (проверка)
```bash
pip install -r requirements.txt
python bot.py
```
Напишите боту `/start` в Telegram — должно появиться меню.

## Как сделать работу 24/7 (бесплатные хостинги)

На домашнем компьютере бот работает только пока включён ПК. Чтобы он работал
круглосуточно, разместите его на бесплатном хостинге. Популярные варианты:

- **Railway** (railway.app) — бесплатные стартовые часы, простой деплой из GitHub.
- **Render** (render.com) — бесплатный тариф для фоновых сервисов / web-сервисов.
- **Replit** (replit.com) — запуск прямо в браузере (для постоянной работы нужен Always-On).
- **PythonAnywhere** (pythonanywhere.com) — бесплатный тариф для постоянно работающих задач.
- **Любой VPS** — если появится возможность.

Общий порядок деплоя:
1. Загрузите папку проекта в репозиторий **GitHub**.
2. На выбранном сервисе создайте новый проект из этого репозитория.
3. В настройках сервиса задайте переменные окружения (те же, что в `.env`):
   `BOT_TOKEN`, `ADMIN_IDS`, `AI_API_KEY`, `AI_BASE_URL`, `AI_MODEL`.
4. Команда запуска: `python bot.py`.

> ⚠️ Никогда не публикуйте файл `.env` и токен в открытом доступе. На хостинге
> используйте переменные окружения, а не файл в репозитории.

## Команды администратора (в чате с ботом)
- `/admin` — панель: число пользователей, сообщений, вопросов, статус ИИ.
- `/stats` — статистика и последние вопросы.
- `/reload` — перечитать `knowledge.md` после правок (без перезапуска).
- `/broadcast текст` — отправить сообщение всем пользователям бота.

## Как менять ответы бота
Откройте `knowledge.md`, отредактируйте нужный раздел (или добавьте контакты в
конце файла) и отправьте боту `/reload`. Новые ответы заработают сразу.

---

## Частые проблемы
- **Бот не отвечает** — проверьте `BOT_TOKEN` и что `python bot.py` запущен без ошибок.
- **Нет умных ответов** — не задан `AI_API_KEY` либо исчерпан лимит ИИ-сервиса.
- **Админ-команды не работают** — ваш Telegram ID не указан в `ADMIN_IDS`.
